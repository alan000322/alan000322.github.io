【ChatGPT系列】ChatGPT如何幫助你提高工作效率？看看那些網友們開發的ChatGPT工具

以下將介紹：

ChatPDF：研究生的福音！快速摘要PDF文件，有問必答！

ChatPDF是一個基於ChatGPT所開發的應用工具，它可以讓使用者將PDF文件上傳至系統中，並且透過問答的方式，輕鬆地尋找想要的資訊。ChatGPT的強大自然語言處理技術，可以讓ChatPDF讀取並理解PDF文件中的內容，進而提供使用者所需要的答案，甚至告訴你所需要的答案的頁碼在哪，提供使用者驗證。

不過使用上，問題還是要盡量問的精確些，才能給予更準確的解答。

ReaderGPT：快速閱讀網站！

ReaderGPT 是一個經由 Google Chrome 與 Microsoft Edge 安裝使用的 Plugin / Extension （擴充功能），可以快速瀏覽網站並寫出摘要。這個擴充功能可以在 Chrome 線上應用程式商店中找到。安裝完成後，使用者可以在擴充功能中找到 ReaderGPT 的功能前，還需要做個人驗證。所謂的個人驗證方式有兩種，一種是直接登入你ChatGPT的帳號，一種是直接使用ChatGPT API 所提供的API key，前者相對比較方便，如果你對後者不熟，可以直接登入ChatGPT帳號就好。
登入完成後，即可開始使用囉！

當瀏覽一個你想要了解的網站，按下Chrome外掛，它就會自動生成一串英文摘要。若你想要呈現繁體中文或輸入其他合適的指令（Prompt），你可以在ReaderGPT中的設定重新進行操作。

bilingual_book_maker：ePUB自動翻譯機

如果你想要閱讀一本電子書，但它還沒有繁體中文版本，那你一定會想要使用這個工具。這個工具是網友自行透過ChatGPT API 串接而成的工具，不過使用門檻有些高，需要在自己的電腦上安裝 Python 3.8（或以上），並自行下指令，且使用API key後token數量（畢竟一本書的文字量龐大）可能遠超過ChatGPT官方給予的免費空間，恐需額外付費，但無論如何換算下來，仍然十分划算。

如果不怕寫程式的朋友，不妨可以試試看。

SQL Chat

SQL Chat是一個經由ChatGPT開發的工具，它可以讓使用者透過一般的語言描述的方式，產生對應用途的SQL語法。

SQL語法的用法對大部分人來說並不直覺，但SQL Chat的使用方法卻是非常直觀且容易上手。只要使用者能夠用語言敘述表達需要查詢的目標，SQL Chat就能快速產生對應的SQL語法，讓使用者不用花費太多時間學習SQL，就能輕鬆查詢資料庫。這種新的方式大幅降低了使用SQL語法的門檻，也讓更多的人能夠快速、準確地查詢到資料庫中的資訊。

趕快來使用看看吧！
